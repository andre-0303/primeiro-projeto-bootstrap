Minhas Skills

HTML

HTML é a base de qualquer página da web. Ele define a estrutura e o conteúdo de um site, organizando elementos como textos, imagens, links e tabelas. Utiliza tags como h1, p, a, img para descrever cada parte do conteúdo. É essencial para criar a base visual de qualquer aplicação web.

CSS

CSS é usado para estilizar páginas HTML. Ele controla cores, fontes, tamanhos e layouts, permitindo criar designs atraentes e responsivos. Com recursos como media queries, é possível adaptar a aparência de um site para diferentes dispositivos, como celulares e desktops.

JavaScript

JavaScript é uma linguagem de programação que traz interatividade para os sites. Ele permite criar animações, validar formulários, manipular elementos HTML dinamicamente e muito mais. É amplamente usado no front-end e também no back-end com tecnologias como Node.js.

React

React é uma biblioteca JavaScript criada pelo Facebook para construir interfaces de usuário. Ele permite criar componentes reutilizáveis e gerenciar o estado da aplicação de forma eficiente. É ideal para desenvolver aplicações web dinâmicas e escaláveis.

Php

PHP é uma linguagem de programação voltada para o back-end, muito usada para criar sites dinâmicos e interagir com bancos de dados. Com ele, é possível construir sistemas de login, gerenciar conteúdo e desenvolver APIs. É altamente compatível com HTML.

MySQL

MySQL é um sistema de gerenciamento de banco de dados relacional. Ele organiza dados em tabelas e permite operações como armazenar, consultar e atualizar informações. É amplamente usado em conjunto com PHP para criar sistemas robustos como lojas virtuais e blogs.
